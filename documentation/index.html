Getting started

Requirements

The requirements for running the Frontend SDK are

  • PHP 5.5+ (Tested with 5.6 - Visit Site)
  • Composer Package manager (Visit Website)

Developer Requirements

  • PHP CLI (Windows CLI)
  • NodeJS (Visit Website)
  • Grunt CLI (Visit Website)
  • Ruby (Visit Website)
  • Compass 0.12.7 (Visit Website)

NOTE: To install compass 0.12.7 you must first uninstall any previous version.

$ gem uninstall compass

And then...

$ gem install compass -v 0.12.7
