Secondary button

Example

Link button

Developer notes

These are the secondary button styles, one applied to a link, the other to a button element. These are created by adding the btn--secondary modifier class to a standard button

<button type="submit" class="btn btn--secondary">Button</button>
<a href="#" class="btn btn--secondary">Link button</a>
