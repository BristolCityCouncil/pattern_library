Primary button

Example

Link button

Developer notes

These are the primary button styles, one applied to a link, the other to a button element.

<button type="submit" class="btn btn--primary">Button</button>
<a href="#" class="btn btn--primary">Link button</a>

Secondary button

Example

Link button

Developer notes

These are the secondary button styles, one applied to a link, the other to a button element. These are created by adding the btn--secondary modifier class to a standard button

<button type="submit" class="btn btn--secondary">Button</button>
<a href="#" class="btn btn--secondary">Link button</a>

Disabled button

Example

Link button

Developer notes

These are the disabled button styles, one applied to a link, the other to a button element. These are created by adding the btn--disabled modifier class to a standard button.

<button type="submit" class="btn btn--disabled" disabled>Button</button>
<a href="#" class="btn btn--disabled" disabled>Link button</a>

Developer notes

This is a download pdf button, it uses an icon and can be primary or secondary (in this case a primary button)
