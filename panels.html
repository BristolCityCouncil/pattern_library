Alert panel

Example

Bad weather delays

We are currently two hours behind schedule for our household collection service. Please be patient.

Usage notes

This panel is used to highlight important situational information which needs to be of the highest priority. This panel is highly interruptive and should be used very sparingly.

<div class="panel panel--alert">
  <div class="ico ico-attention ico--large"></div>
  <h2 class="panel__title">Bad weather delays</h2>
  <div class="panel__copy">
    <p>We are currently two hours behind schedule for our household
      collection service. Please be patient.</p>
  </div>
</div>

Create panel

Example

Create a new account

It's easy and takes about 2 minutes

Developer notes

This panel is an action panel, which should consist of a title, button and supporting text. The background is achieved by adding the panel--primary modifier, and the button colour is achieved by using a primary button

Usage notes

Bespoke element to be used only on the Sign in / Create account choice page, or any other page where creating a new account is a major call-to-action.

<div class="panel panel--action panel--primary">
  <div class="panel__inner">
    <div class="panel__vertical-centre">
      <h3 class="panel__title">Create a new account</h3>
      <p>
        <button class="btn" type="submit">Register now
          <span class="ico ico-right"></span>
        </button>
      </p>
      <p>It's easy and takes about 2 minutes</p>
    </div>
  </div>
</div>
