Single address returned

Example

Developer notes

This is the implementation for the address lookup when a single address is returned after being submitted

<div class="form__item form__item--string-action">
  <label class="form__label">Your postcode</label>
  <input class="form__field form__field--string" type="text" name="" value="BS6 5AA"
  />
  <button class="btn btn--secondary" type="submit">Lookup address</button>
</div>
<!--/form__item-->
<div class="form__item form__item--select">
  <select name="" class="form__field form__field--select">
    <option>Please select...</option>
    <option selected="selected">19 Redland Road, Bristol, BS6 5AA</option>
  </select>
</div>

Multiple address returned

Example

Developer notes

This is the implementation for the address lookup when multiple addresses are returned after being submitted, the selected address panel is rendered with JS

<div class="address-lookup js-address-lookup">
  <div class="form__item form__item--string-action">
    <label class="form__label">Your postcode</label>
    <input class="form__field form__field--string" type="text" name="" value="BS6 5AA"
    />
    <button class="btn btn--secondary" type="submit">Lookup address</button>
  </div>
  <!--/form__item-->
  <div class="form__item form__item--select">
    <select name="" class="form__field form__field--select form__field--selectsize"
    size="3">
      <option selected="selected">Please select...</option>
      <option>19 Redland Road, Bristol, BS6 5AA</option>
      <option>20 Redland Road, Bristol, BS6 5AA</option>
      <option>31 Redland Road, Bristol, BS6 5AA</option>
    </select>
  </div>
</div>
